# Web_Monitor
Un sistema de monitoreo que:  Analiza los logs de Apache para detectar IPs con actividad sospechosa.

Identifica posibles ataques como fuerza bruta o escaneo masivo de directorios.

Envía alertas automáticas a Telegram a través de un bot personalizado.

Permite bloquear y desbloquear IPs directamente desde una interfaz web con Flask.

Integrar dashboards en tiempo real con estadísticas.

Correlacionar eventos con otras fuentes de logs del sistema.

# Configuracion

Ir al archivo config.js y anexar la API_KEY y ID de telegram.

# Instalacion

pip install -r requirements.txt

# Activacion

Ir al Directorio backend y python3 app.py
